script脚本命令

  • npm自己的生命周期命令：可发着写的钩子（如preinstall, postinstall, prepublish, postpublish等……）
  • 自定义的命令：
    • “dev”: “webpack-dev-server” 在npm run dev时原地启动， 这是个开发服务器

npm install の过程：

  1. 寻找包版本信息文件（package.json）,依赖它进行安装
  2. 查package中的依赖，并检查项目中其他版本文件
  3. 如果发现新的包，更新版本信息文件
